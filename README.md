# Simple E-commerce Store

This is a simple full-stack e-commerce web application built as part of the CodeAlpha internship task.

## Features
- User Registration & Login
- Product Listing with Categories
- Shopping Cart (Add / Remove / Quantity)
- Order Placement
- MongoDB Database Integration

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start server using `node server.js`
4. Open frontend files in browser
